# ECO RECYCLING & INJECTION METAL

Modern industrial website for a premier metal recycling company based in Zaghouan, Tunisia.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4
- **Components**: shadcn/ui
- **Icons**: Lucide React
- **Theme**: Custom Teal-Green (`#006b3c`) and Coffee (`#faf0e6`) palette

## Project Structure

- `app/`: Main application routes and pages
  - `page.tsx`: Homepage with hero section, dramatic stats, and "Why Choose Us"
  - `products/page.tsx`: Catalog of Ferrous and Non-ferrous metals with stock availability
  - `about/page.tsx`: Detailed overview of Warehousing, Shipping, Operations, and Quality
  - `contact/page.tsx`: Contact information and location details
  - `globals.css`: Global styles and Tailwind theme configuration
- `components/`: Reusable React components
  - `navbar.tsx`: Sticky navigation with logo and page links
  - `footer.tsx`: Company information, professional memberships, and LinkedIn link
- `public/`: Static assets including images and the company logo
- `lib/`: Utility functions

## Key Features

- **Global Recognition**: Prominent display of ReMA, CMRA, and BIR memberships.
- **Dynamic Stats**: Interactive counters showing global operations and representatives.
- **Mobile First**: Fully responsive design optimized for all device sizes.
- **Accessibility**: Semantic HTML and ARIA attributes for better accessibility.

## Design System

- **Primary Green**: `#006b3c` (Used for brand identity and primary actions)
- **Coffee Shade**: `#faf0e6` (Used for section backgrounds and feature cards)
- **Typography**: Optimized Geist Sans font for readability and industrial feel.
